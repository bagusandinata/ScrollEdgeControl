# ScrollEdgeControl
Yet another UIRefreshControl

## Installation

**Cocoapods**

Including custom activity indicator
```ruby
pod "ScrollEdgeControl"
```

If you need only core component
```ruby
pod "ScrollEdgeControl/Core"
```

**SwiftPM**
